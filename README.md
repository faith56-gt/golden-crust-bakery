# golden-crust-bakery
A fully responsive multi-page website for Golden Crust Bakery, built with HTML5, CSS3, JavaScript, and Bootstrap 5.

## Pages
- Home
- About
- Products/Menu
- Gallery
- Blog
- FAQ
- Contact
- Testimonials

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5

## Features
- Responsive design (mobile, tablet, desktop)
- Contact form with validation
- Dark/Light mode toggle
- Filterable product gallery
